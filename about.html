ABOUT THE PROJECT

Pathways is a creative data project showing a month's worth of geolocation data and message metadata collected from the mobile phones of Londoners. The aim of Pathways is to show what digital data looks like in the context of common relationships. As the number of internet-enabled devices increases, we generate more and more data that is intimately and invisibly connected to us. This is the reality within which Pathways is situated.

All participants willingly volunteered their data; data was retained solely for the duration of the project. Participants were found through in-person recruitment and email open calls.

This project was created by Mimi Onuoha as part of the 2014-15 Fulbright-National Geographic Digital Storytelling Fellowship. To learn more about the development process and the themes that inspired the project, see the project posts on National Geographic's blog. To learn more about Mimi and her work, visit her personal website.

For more on the Fulbright National Geographic Digital Storytelling Fellowship, visit the Fulbright site.

Data & Methodology

All data gathered between November 2014 and March 2015. Geolocation data tracked using OpenPaths and Moves. All code for this site is available on Github.
